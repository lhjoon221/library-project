# 📚 Library Info Hub
도서 검색 및 도서관 이용 정보를 제공하는 웹 서비스입니다.

## 🛠 기술 스택
- HTML5, CSS3, JavaScript
- **Supabase**: 데이터베이스 연동 및 게시판 기능 구현
- **Kakao Maps API**: 도서관 위치 시각화

## 📌 주요 특징
- 웹 표준과 시맨틱 마크업을 준수하여 검색 엔진 최적화(SEO)를 고려하였습니다.
- 외부 API와 DB를 연동하여 실제 서비스 가능한 수준의 데이터 처리 로직을 구축하였습니다.
